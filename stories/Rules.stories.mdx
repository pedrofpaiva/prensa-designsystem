# FAMÍLIAS

### Como funcionam as famílias

O prensa está dividido em famílias globais e famílias específicas. Uma família é montada no SCSS com tokens e propriedades css e executa a suas possíveis configurações utilizando as `props` de um componente no javascript. Cada **família** contém seu conjunto de **tokens** que carregam os valores do css. As famílias globais estão disponíveis em SCSS e em formato JSON para serem utilizadas em qualquer componente do projeto. Já as específicas são famílias que ainda não foram desenvolvidas o suficiente ou pertencem a contextos específicos - um bom exemplo é a família de tamanhos de botão, que pertence somente ao componente Button.

O **ponto de encontro** entre componentes e famílias é através do seu `class` onde entra a atuação do JavaScript. Cada class irá corresponder a um conjunto de estilos, ou uma família. Um class no prensa é montado através de um conjunto de props de um componente e tem a possibilidade de um sufixo customizado para casos específicos.

Abaixo temos como exemplo o class do componente Button

<pre>
  <CodeBlock
    language="scss"
    children={
    `
    .Prensa-Button-root {
      @include px($space-size-2);
      @include miw(max-content);
      @extend %family-buttonSize;
      @extend %family-radius;
      cursor: pointer;
      border: unset;
      text-transform: uppercase;
    }
    `
    }
  />
</pre>

### Construção das classes

Sempre trabalharemos com classes no padrão de escrita do Prensa:

**`Prensa- + ${componente}- + ${prefixo}- + ${famílias} + ${sufixo}`**

**`Prensa-Button-root color-primary-1`**

> (produto) Prensa: identifica o design system na DOM.

> (componente) Button: nome do componente sempre com a letra maíuscula

> (prefixo) root: é referente ao elemento do componente que acessamos. Pode ser root (raíz), ou icon por exemplo

> color-primary-1: é referente a cor escolhida da família de cores

No SCSS o valor lido no className será interpretado na família de cores, como podemos ver abaixo. O `class` que executaria o trecho de código SCSS abaixo seria `Prensa-Button-root color-primary-1`, com as `props` sendo **`color='primary-1'`** e os outros valores default

<CodeBlock language='scss'>
  {`
    %family-background-colors {
      &.color-primary-1 {
        background-color: $color-primary-1;
      }
    }
  `}
</CodeBlock>
