### Variações
O Button carrega duas variações de estilo, **Filled** e **Outlined**. Elas alteram a escolha dos tokens para o esquema de cores. 

<Button variant="filled">Avançar</Button>
<Button variant="outlined">Avançar</Button>

<Space size={2} />

<pre>
  <CodeBlock children={`
  <Button variant="filled">Avançar</Button>
  <Button variant="outlined">Avançar</Button>
  `}/>
</pre>


<Space size={6} />

### Esquema de cores
O componente possui um esquema de cores pré-definidas, construídas com base no tema do design-system. Usamos a prop `color` para a escolha entre `color="primary"` e `color="secondary"`.

<Button variant="filled" color='primary'>Primário</Button>
<Button variant="filled" color='secondary'>Secundário</Button>

<Space size={2} />

<pre>
  <CodeBlock children={`
  <Button variant="filled" color="secondary">Avançar</Button>
  <Button variant="outlined" color="secondary">Avançar</Button>
  `}/>
</pre>

<Space size={6} />

### Funcionalidades
O componente pode receber uma prop chamada `onClick` que permite a passagem da ação do clique realizado pelo usuário. Também pode receber a prop `fullWidth` que faz com que o componente assuma o formato 100% para o estilo.

<Space size={2} />

<pre>
  <CodeBlock children={`
  import Button from 'prensa/Button';

  const handleClick = () => {
    console.log('click')
  }

  const Component = props => {
    return <Button variant="filled" color="secondary" onClick={handleClick()}>Avançar</Button>;
  }
  `}/>
</pre>
