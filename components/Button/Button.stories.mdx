### Variações
O Button carrega três variações de estilo, **filled**, **outlined** e **ghost**. Elas alteram a renderização da estrutura do botão para diferentes casos de uso. Para filled, usamos background-color. Já outlined, usa border-color com backgorund transparente. O ghost não utiliza bordas e tem um background transparente.

<ThemeProvider theme={theme}>
  <div style={{display: 'flex'}}>
    <Button variant="filled" style={{marginRight: 8}}>Avançar</Button>
    <Button variant="outlined" style={{marginRight: 8}}>Avançar</Button>
    <Button variant="ghost" style={{marginRight: 8}}>Avançar</Button>
  </div>
</ThemeProvider>

### Esquema de cores
O componente pode receber na sua prop `color` todos os tokens de cor existentes dentro do theme.

Existem três formas de utilizar a prop color:

<Space size={4} />

> A. Color receber o token de cor sem o prefixo color ou hífen: `color="editorial3"`

<ThemeProvider theme={theme}>

  </div>
</ThemeProvider>

<Space size={2} />

<pre>

</pre>

<Space size={4} />

> B. Color receber um valor do theme através das props: `props.theme.colors.primary1`

  <div style={{display: 'flex'}}>

</ThemeProvider>

### Border radius
O componente possui acesso a família de variações para o css border-radius. Podemos alternar entre um botão quadrado e dois níveis de radius. A props é `radius` e as opções são `false`, `default` e `alternative`.

<ThemeProvider theme={theme}>
  <div style={{display: 'flex'}}>
    <Button variant="filled" color='primary1' radius={false} style={{marginRight: 8}}>False</Button>
    <Button variant="filled" color='primary1' radius='default' style={{marginRight: 8}}>Default</Button>
    <Button variant="filled" color='primary1' radius='alternative' style={{marginRight: 8}}>Alternative</Button>
  </div>
</ThemeProvider>

### Tamanhos
O componente possui acesso as duas bibliotecas de tamanho do prensa, formato **Number** e **String**.

Passando um **Number** para a props size o tamanho será equivalente ao valor `size * theme.factors.margin`.

Utilizando o valor de size **String** o retorno será exatamente o tamanho especificado em pixels.

`size="42px" | 42px`
`size={4} | 32px`

<ThemeProvider theme={theme}>
  <div style={{display: 'flex', alignItems: 'center'}}>
    <Button variant="filled" color='primary1' size="42px" style={{marginRight: 8}}>Entrar</Button>
    <Button variant="filled" color='primary1' size={4} style={{marginRight: 8}}>Entrar</Button>
  </div>
</ThemeProvider>

### Estados
O componente possui dois estados: `disabled` e `hover`. Para acionar o disabled, deve-se utilizar a props `disabled={true}`. Já o hover aciona automaticamente ao passar o mouse sobre o botão. O estilo de ambos é sempre fixo.

<ThemeProvider theme={theme}>
  <div style={{display: 'flex'}}>
    <Button variant="filled" color='primary1' disabled={true} style={{marginRight: 8}}>Disabled</Button>
    <Button variant="outlined" color='primary1' disabled={true} style={{marginRight: 8}}>Disabled</Button>
    <Button variant="ghost" color='primary1' disabled={true} style={{marginRight: 8}}>Disabled</Button>
  </div>
</ThemeProvider>

### Funcionalidades
O componente pode receber uma prop chamada `onClick` que permite a passagem da ação do clique realizado pelo usuário. Também pode receber a prop `fullWidth` que faz com que o componente assuma o formato 100% para o estilo.

<Space size={2} />

<pre>
  <CodeBlock children={`
  import {Button} from 'prensa';

  const handleClick = () => {
    console.log('click')
  }

  const Component = props => {
    return <Button variant="filled" color="secondary" onClick={handleClick()}>Avançar</Button>;
  }
  `}/>
</pre>

<Space size={10} />

### Botão com ícone
O componente pode receber ícones do lado esquerdo ou direito através das props `leftIcon` e `rightIcon`, respectivamente. Para utilizá-las basta importar um ícone SVG como componente e passar o mesmo na props.

<ThemeProvider theme={theme}>
  <div style={{display: 'flex'}}>
    <Button variant="filled" color='primary1' leftIcon={<DownloadIcon />} style={{marginRight: 8}}>Com ícone</Button>
    <Button variant="filled" color='primary1' rightIcon={<ArrowRightIcon />} style={{marginRight: 8}}>Com ícone</Button>
    <Button fullWidth variant="ghost" color='primary1' rightIcon={<ArrowRightIcon />}>Com ícone variação ghost</Button>
  </div>
</ThemeProvider>
