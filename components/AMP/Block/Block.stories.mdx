# Block

<Space size={3} />

Block é um componente baseado em uma `<div>` para entregar posicionamentos blocados na tela.

<Space size={6} />

### Como utilizar

<Space size={3} />

<pre>
  <CodeBlock>{`import Block from 'prensa/amp'`}</CodeBlock>
  <CodeBlock>{`<Block>children</Block>`}</CodeBlock>
</pre>

<Space size={6} />

### Propriedades de alinhamento

<Space size={3} />

Utilize a propriedade `align` para alternar entre o flex `row` e `col`

Utilize a propriedade `alignx` para definir a posição no eixo x

Utilize a propriedade `aligny` para definir a posição no eixo y

Utilize a propriedade `alignz` para definir o alinhamento interno

<Space size={6} />

### Exemplo 1: 

`row left top`

<pre>
  <CodeBlock>{`<Block align='row' alignx='left' aligny='top'></Block>`}</CodeBlock>
</pre>

<Space size={3} />

<Block align='row' alignx='left' aligny='top' theme='demo'>
  <span>item1</span>
  <span>item2</span>
  <span>item3</span>
</Block>

### Custom styles

Para adicionar estilos customizados utilize `custom`

<pre>
  <CodeBlock>{`<Block custom='
    background-color: rgba(1, 22, 39, 0.8); 
    color: rgba(255, 255, 255, 1);
    span {
      border: 1px solid rgba(55, 255, 255, 1);
    }
  '></Block>`}</CodeBlock>
</pre>
